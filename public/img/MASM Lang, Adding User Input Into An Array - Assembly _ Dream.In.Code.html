Can you do me a small favor TechDee... Respond to my replies to your posts, say something like - thanks, got it, ok, etc just so I know you read and understand my replies. I replied and you ask almost the same exact question again.

What is line 16? If you are going to add 4 then say add esi, 4.

You want an array that holds 2 dwords?
SomeArray      DWORD 2 dup (?)


to add to the array:
    mov     esi, offset SomeArray
    mov     dword ptr [esi], 2 ; move 2 into first 4 bytes of array
    mov     dword ptr [esi + 4], 3 ; mov 3 into last 8 bytes of array


to retrieve:
    mov     eax, [esi] ; get number at index 0 of array (first 4 bytes)
    call    WriteInt
    call    Crlf
    
    mov     eax, [esi + 4] ; get number at index 1 of array (last 8 bytes)
    call    WriteInt
    call    Crlf    
    call    Crlf

    ; or to use in a loop
    xor     ecx, ecx
@@:
    mov     eax, [esi + 4 * ecx]  
    call    WriteInt
    call    Crlf
    inc     ecx
    cmp     ecx, 2
    jne     @B


every time you do:
mov [esi],eax
you are overwriting what is in the first 4 bytes of the array.
